# Aggie Fundraisers 

A centralized hub for clubs at UC Davis to upload fundraising events, allowing students to easily browse nearby fundraisers while downtown and save money on purchases (if possible) while supporting campus causes.

Made with Next.js, Node.js, Express, and MongoDB. 

Check it out at: [https://aggie-fundraisers.vercel.app/](https://aggie-fundraisers.vercel.app/)
