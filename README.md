# Aggie Fundraisers 

A centralized hub for organizations at UC Davis to upload fundraising events, allowing students to easily browse nearby fundraisers while downtown and save money on purchases while supporting campus causes.

Made with Next.js, Node.js, Express, and MongoDB.

Check it out at: [https://aggie-fundraisers.vercel.app/](https://aggie-fundraisers.vercel.app/)

Will be updating the UI soon, stay tuned!
